# Data Analyst Internship - Task 2

## Objective

To create visualizations that convey meaningful insights using sales data.

## Tools Used

* Power BI

## Dataset

* Superstore Sample Dataset

## Visualizations

* Sales by Region
* Profit by Category
* Sales Trend Over Time
* Sales by Segment

## Key Insights

* West region has the highest sales
* Technology category is the most profitable
* Consumer segment contributes the highest sales
* Sales show fluctuations over time

## Conclusion

This dashboard helps in understanding sales performance, identifying trends, and making better business decisions.
